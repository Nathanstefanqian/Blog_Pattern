# 1.1 二维图像模式匹配介绍

在计算机科学中，二维图模式匹配问题（2D pattern matching）是指在一个二维的文本图像中查找一个给定的模式图像是否存在的问题。在这个问题中，文本图像被表示为一个二维矩阵，而模式图像同样也被表示为一个二维矩阵。该问题在计算机视觉、图像处理、模式识别等领域中有着广泛的应用，如人脸识别、指纹识别、图像检索等

例如，在给定图1（a）所示的二维文本图像中查找特定的模式图像图1（b），主要处理的过程为：首先把二维文本图像和给定的模式图像转换为二维矩阵，对图像中的每一个像素点采样，得到对应的采样值，存储为二维矩阵，这样就得到 二维文本图像和给定的模式图像的二维矩阵形式；然后问题转化为在文本图像表示的大矩阵中查找给定的模式图像的小矩阵，然后选择合适的匹配算法进行查找匹配。

图1(a)文本图像
图1(b)模式图像

因此，二维图像模式匹配的关键问题在于设计合适的匹配算法。在处理大规模图像时，匹配算法需要能够高效地处理大量数据，并且需要考虑存储和计算的问题。如何设计高效的算法和数据结构，以提高匹配的速度和可扩展性，是一个重要的问题。

# 1.2 问题简化

为了更加清楚说明二维图像模式匹配算法问题，对问题进行简化，即假设图像的每一个像素点只有黑白像素两种颜色，在二维矩阵中用0表示黑色，用1表示白色。对于更多种颜色的二维图像也很容易扩展类比。

因此，对于二维图模式匹配的算法研究可以简化表述为：现有如图2（a）所示的文本矩阵和如图2（b）所示的模式矩阵，现在需要在文本矩阵中查找是否存在和模式矩阵相同的子矩阵，如果找到则成功，否则失败。因此，问题关键在于查找算法的设计。
